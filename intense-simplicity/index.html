This template is my first submission to OWD, and my first design created for the sake of being a template used by others :). Enjoy!

This template is easy to use, taking advantage of CSS and the reusability of styled components. I tried to style as many elements as I could, but I probably missed a few things.

If you don't like the header above. Just stick one of them in the [template root] / images / headers / directory and then go into the CSS file and replace the text "h1.jpg" with the name of the header file.

"css/default.css" - about line 258
#header{
  background:url(../images/headers/h1.jpg) no-repeat top center;
}

I would love to hear how this template is used, so if you use it, please send me an email and let me know!
